# 服务端任务拆解

## 1. 目标

实现 Mac 服务端 MVP，提供屏幕 MJPEG 流、控制 JSON API、输入映射、快捷文本、快捷按钮和快捷命令执行能力。

## 2. 工作目录

所有服务端代码只能写入：

- `server-mac/`

共享协议只读参考：

- `docs/共享协议与MVP边界.md`

## 3. 技术要求

- SwiftPM 项目。
- 可执行产品名：`VibeLinkServer`。
- 默认监听端口：`8765`。
- 支持启动参数：`--port`、`--token`。
- 不依赖 GUI app bundle。
- 尽量使用系统库，避免大型第三方依赖。

## 4. 功能任务

### 4.1 项目脚手架

- 创建 `Package.swift`。
- 创建 `Sources/VibeLinkServer/`。
- 创建可执行入口。
- 创建基本单元测试或可运行 smoke test。

### 4.2 HTTP 服务

- 实现基础 HTTP server。
- 支持 `GET /health`。
- 支持 `GET /stream?token=<token>`。
- 支持 JSON API 路由。
- 所有受保护 API 校验 token。
- 对无效路径返回 404，对鉴权失败返回 401。

### 4.3 屏幕采集与 MJPEG

- 实现 JPEG 截屏。
- MVP 可调用 `/usr/sbin/screencapture -x -t jpg -` 获取 JPEG bytes。
- `GET /stream` 以 multipart MJPEG 方式循环写帧。
- 默认帧率可设为 3-5 FPS，先保证稳定。
- 截屏失败时返回可读错误或断开流。

### 4.4 输入映射

- `tap`：CGEvent 鼠标单击。
- `doubleTap`：两次鼠标单击。
- `rightClick`：右键点击。
- `drag`：鼠标按下、移动、抬起。
- `scroll`：滚轮事件。
- `text`：写入剪贴板后模拟 Cmd+V；`submit=true` 时追加回车。
- `clipboard`：只写入剪贴板。

### 4.5 快捷文本

- 内置至少三条快捷文本：
  - 继续修复。
  - 运行相关测试。
  - 总结当前修改。
- `GET /api/quick-texts` 返回列表。

### 4.6 快捷按钮

- MVP 内置至少一个示例快捷按钮：点击屏幕中央。
- `GET /api/shortcut-buttons` 返回列表。
- `POST /api/shortcut-buttons/run` 执行按钮。
- 预留后续从配置文件读取的结构。

### 4.7 快捷命令

- 内置至少三条命令：
  - `pwd`
  - `git status --short`
  - `ls`
- `GET /api/commands` 返回列表。
- `POST /api/commands/run` 异步执行命令并返回 `runId`。
- `GET /api/commands/runs/<runId>` 返回状态、输出、退出码。
- 命令必须限定为预设列表。

### 4.8 日志与启动提示

- 启动时打印：
  - 监听地址。
  - token。
  - health URL。
  - stream URL。
  - macOS 权限提示。
- 请求错误和命令执行结果打印到控制台。

## 5. 验收

必须支持：

```bash
cd server-mac
swift build
swift run VibeLinkServer --port 8765
```

服务启动后：

```bash
curl http://127.0.0.1:8765/health
curl -H "Authorization: Bearer <token>" http://127.0.0.1:8765/api/commands
```

验证后返回：

- 修改了哪些文件。
- 构建是否通过。
- health 是否可访问。
- API 是否可访问。
- 仍未完成的风险。
